# Electronics Store Web Application

A lightweight e-commerce web application built using **Flask (Python)** for managing and browsing an electronics store inventory.

---

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **Database:** MongoDB (NoSQL)

---

## 🚀 How to Run Locally

### 1. Clone the Repository
```bash
git clone [https://github.com/Zohaib-Khan-Chandio/electronics-store.git](https://github.com/Zohaib-Khan-Chandio/electronics-store.git)
cd electronics-store



### 2. Install Dependencies
```
pip install -r requirements.txt
