# Electronics Store Web Application

A lightweight e-commerce web application built using **Flask (Python)** for managing and browsing an electronics store inventory.

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **Database:** MongoDB (NoSQL)

---

## 🚀 How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/Zohaib-Khan-Chandio/electronics-store.git
cd electronics-store
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Setup MongoDB 🍃

Make sure you have MongoDB Compass installed locally, or use MongoDB Atlas (Cloud).

Ensure your MongoDB server is running on:

```bash
mongodb://localhost:27017/
```

(Optional) Populate the database with sample data:

```bash
python seed.py
```

### 4. Start the Flask Server

```bash
python app.py
```

### 5. Open in Browser

Visit the following URL in your browser:

```text
http://127.0.0.1:5000/
```

The application should now be running locally.
