# Electronics Store Web Application

A lightweight e-commerce web application built using **Flask (Python)** for managing and browsing an electronics store inventory.

---

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **Database:** MongoDB (NoSQL)

---

## 🚀 How to Run Locally

### 1. Clone the Repository
```bash
git clone [https://github.com/Zohaib-Khan-Chandio/electronics-store.git](https://github.com/Zohaib-Khan-Chandio/electronics-store.git)
Bash
cd electronics-store
2. Install Dependencies
Bash
pip install -r requirements.txt
3. Setup MongoDB 🍃
Make sure you have MongoDB Compass installed locally, or you are using MongoDB Atlas (Cloud).

Ensure your local MongoDB server is running on mongodb://localhost:27017/.

(Optional) Run the seed script to populate initial data:

Bash
python seed.py
4. Start the Flask Server
Bash
python app.py
Open http://127.0.0.1:5000/ in your browser to view the application.
